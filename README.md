# KaySquadron

**KaySquadron** est un plugin Dalamud pour Final Fantasy XIV conçu pour vous aider à gérer et optimiser les entraînements de votre escouade de Grande Compagnie.

## Fonctionnalités

- **Calculateur d'optimisation** : Sélectionnez une mission, et le plugin calculera instantanément toutes les combinaisons possibles pour atteindre les prérequis (100% de réussite), en vous indiquant exactement quel entraînement physique, mental ou tactique vous devez effectuer au tableau pour y arriver !
- **Importation de l'escouade actuelle** : Scanne la mémoire du jeu (GcArmyManager) pour importer automatiquement le nom, la classe et le niveau exacts de vos 8 membres actuels.
- **Thème UI Personnalisé** : Interface propre et réactive arborant le thème **Kay Cyan**, unifiant l'esthétique avec d'autres plugins comme KayDps et KayArsenal.
- **Multilingue** : Support complet du Français et de l'Anglais, basé sur la langue d'affichage du jeu ou de Dalamud.

## Installation

Ce plugin est disponible via le dépôt personnalisé KayRepo.

1. Ouvrez les paramètres de Dalamud en jeu (`/xlsettings`).
2. Allez dans l'onglet **Expérimental**.
3. Dans la section "Dépôts de plugins personnalisés" (Custom Plugin Repositories), ajoutez l'URL de KayRepo :
   `https://raw.githubusercontent.com/KayrozDbs/KayRepo/main/pluginmaster.json`
4. Cliquez sur le bouton `+` puis sauvegardez.
5. Recherchez **KaySquadron** dans la liste de tous les plugins (`/xlplugins`) et installez-le.

## Utilisation

- Utilisez la commande en jeu `/ksquad` pour ouvrir ou fermer l'interface de KaySquadron.
- Allez dans l'onglet **Membres** et cliquez sur "Importer de l'Escouade actuelle" une fois rendu dans votre caserne de Grande Compagnie.
- Allez dans l'onglet **Optimisation**, choisissez votre mission critique à la table des missions, et suivez les instructions d'entraînement sur le tableau de bord !

## Prérequis

- **Final Fantasy XIV : Dawntrail**
- **Dalamud** (API v14+)
